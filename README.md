## Installation

```bash
git clone https://github.com/nohackjustnoobb/Mark-Six-Simulator.git
```

```bash
cd Mark-Six-Simulator
```

```bash
npm install
```

## Run

```bash
npm start
```

## Build

```bash
npm run build
```
